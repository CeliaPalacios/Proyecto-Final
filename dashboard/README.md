# 📊 Dashboard Power BI

Este dashboard complementa el análisis exploratorio realizado en el proyecto, permitiendo visualizar de forma interactiva los principales resultados.

## 🧩 Estructura

El dashboard se divide en tres páginas:

* **Resumen general** → KPIs y visión global del dataset
* **Análisis por género y país** → patrones estructurales
* **Rendimiento y popularidad** → relaciones entre variables
